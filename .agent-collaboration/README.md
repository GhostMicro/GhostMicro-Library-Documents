# AI Collaboration Hub

ศูนย์กลางการสื่อสารระหว่างคุณ (User) และผม (Antigravity AI) เพื่อให้การทำงานร่วมกันเป็นไปได้อย่างต่อเนื่องและมีการบันทึกที่ชัดเจน

## โครงสร้างในโฟลเดอร์นี้
- `Work_Log.md`: บันทึกการทำงาน ประวัติ และความคืบหน้าของโปรเจกต์
- `Technical_Decisions.md`: บันทึกการตัดสินใจทางเทคนิคและสถาปัตยกรรมที่สำคัญ
- `Resources.md`: (ถ้ามี) รวบรวมลิงก์หรือไฟล์สำคัญที่เกี่ยวข้อง

## กฎสำคัญ (Critical Rules)
- **ห้ามลบไฟล์** ในโฟลเดอร์นี้เด็ดขาด (No deletion of files)
- **อนุญาตให้แก้ไข** อัปเดต หรือเพิ่มเติมข้อมูลได้ตลอด (Updates and additions only)

## วิธีใช้งาน
- ผมจะทำการอัปเดตไฟล์เหล่านี้เป็นระยะเมื่อมีการเปลี่ยนแปลงสำคัญ
- คุณสามารถเข้ามาอ่านเพื่อดูภาพรวมหรือทิ้งโน้ตไว้ให้ผมได้ที่นี่ครับ
