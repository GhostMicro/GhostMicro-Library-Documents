# Technical Decisions

บันทึกการตัดสินใจเชิงเทคนิคและสถาปัตยกรรม

## [2026-01-06] - การจัดระเบียบเอกสารสื่อสาร
- **หัวข้อ**: การใช้ Hidden Folder (`.agent-collaboration`)
- **บริบท**: ผู้ใช้ต้องการพื้นที่ส่วนกลางสำหรับบันทึกงานโดยเฉพาะ
- **การตัดสินใจ**: ใช้ชื่อโฟลเดอร์ที่ขึ้นต้นด้วยจุดเพื่อให้แยกแยะจากเอกสารตัวโปรเจกต์หลักได้ชัดเจน แต่ยังอยู่ใน Workspace เพื่อความสะดวกในการเข้าถึง
- **สถานะ**: อนุมัติและดำเนินการแล้ว

## [2026-01-06] - การเปลี่ยน Default Port สำหรับ MkDocs
- **หัวข้อ**: การใช้ Port 8081 แทน 8000
- **บริบท**: Port 8000 มักถูกใช้งานโดยโปรเซสอื่นในระบบ ทำให้ MkDocs รันไม่ได้
- **การตัดสินใจ**: กำหนดให้ใช้ Port 8081 เป็นมาตรฐานสำหรับโปรเจกต์นี้ และสร้างสคริปต์ `serve_docs.sh` เพื่อช่วยให้รันได้ง่ายขึ้น
- **สถานะ**: อนุมัติและดำเนินการแล้ว
